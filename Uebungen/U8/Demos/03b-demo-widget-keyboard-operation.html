Widget Keyboard Operation

Radio-Button Group with one Element Selected

Radio Buttons: When the radio-button-group (same name) receives focus the checked button in group receives the actual fokus then left/right cursor key move check. When no element in the group is yet checked the first radio button is focused but not yet checked. The space-key then checks the focused element. Also using the cursor key will check the item selected next. Return key might activate the forms submit button.

Checkboxes

Always first element in group will receive focus, tab will then move to the next. Space-key will check/uncheck the currently focused checkbox. Cursor-keys will have no effect. Return key might activate the default button.

What ice cream flavors do you like?

Text Input Fields

Always the first text-input will receive focus, tab will then move to the next. Cursor-keys will move within text (if applicable). Return key might activate the default button.

Buttons

Always the first button will receive focus, tab will then move to the next. Space-key and return-key will activate button with current focus. Cursor-keys will have no effect.

Drop Down

When focused a drop-down is opened with cursor up/down keys. Current item from drop-down is selected with return-key or space-key
