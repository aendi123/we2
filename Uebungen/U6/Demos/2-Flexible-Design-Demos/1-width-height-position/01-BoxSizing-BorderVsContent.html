Value of box-sizing: content-box vs. border-box

  • .box div (gray background, 10px inner padding, 10px coral border, 10px margin) is sized at 70x70px
  • .container around the .box has a lightgreen background
  • first square has the default box-sizing: content-box (width & height determine size of content, padding+border+margin go on top)
  • second square has box-sizing: border-box (width & height determine size of bounding box of content+padding+border)
  • the text 70x70 appears with padding in the gray area (padding cannot hava specific color)
  • there is no defined standard box-sizing: margin-box
  • Border Box is helpful if a given space should be used (e.g. 3 boxes in line each using 33.3%)
  • Content Box is helpful if the size of the content is known
